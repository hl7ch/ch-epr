# CH Term Implementation Guide
FHIR® implementation guide for Swiss terminology used in the CH Core profiles and for different exchange formats, see [fhir.ch](https://fhir.ch/).

## Swiss EPR Context
This implementation guide contains also the metadata specified in the framework of Annex 3 and Annex 9 of the [FDHA Ordinance](https://www.bag.admin.ch/bag/de/home/gesetze-und-bewilligungen/gesetzgebung/gesetzgebung-mensch-gesundheit/gesetzgebung-elektronisches-patientendossier.html) on the electronic patient record (EPR) in Switzerland.

This artefacts are based on the [CH-EPR - Electronic Patient Record repository](http://ehealthsuisse.art-decor.org/index.php?prefix=ch-epr-) [ART-DECOR®](https://www.art-decor.org/mediawiki/index.php/Main_Page) which has been [published by eHealth Suisse](http://ehealthsuisse.art-decor.org/).
